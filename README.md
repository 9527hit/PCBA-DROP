# PCBA跌落失效微观演示

这是一个基于Web的交互式演示，展示了PCBA在跌落过程中BGA焊点和PCB的微观失效机理。

## 如何运行

### 方法 1: 直接打开
直接双击文件夹中的 `index.html` 文件，在浏览器中打开即可。

### 方法 2: 使用 Python 服务器 (推荐)
在当前目录下打开终端 (PowerShell 或 CMD)，运行以下命令：

```bash
python -m http.server 8000
```

然后在浏览器中访问: `http://localhost:8000`

## 功能说明

*   **宏观视图**: 展示PCB板在跌落冲击下的弯曲变形。
*   **微观视图**: 展示BGA焊点内部的应力集中和断裂过程。
*   **失效模式选择**:
    *   **IMC层脆性断裂**: 焊料与焊盘之间的金属间化合物层断裂。
    *   **焊盘坑裂 (Pad Cratering)**: PCB基材树脂被撕裂，焊盘被拔起。
*   **物理过程解说**: 实时文字说明当前发生的物理现象（如应变率硬化、裂纹成核等）。
